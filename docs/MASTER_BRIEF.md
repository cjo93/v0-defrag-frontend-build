# DEFRAG — BUILD BRIEF FOR JULES

## 1. PRODUCT DEFINITION
DEFRAG is a private AI system that helps users:
* Understand what is happening in tense interactions
* Know when it is a good time to talk
* Respond safely and clearly
* Recognize repeating relationship patterns
* Reduce escalation

It supports:
* Individuals, Couples, Families, Small teams

It does not replace therapy. It does not diagnose. It does not predict destiny. It provides timing-aware guidance and communication support.

## 4. AI CHAT (PRIMARY FEATURE)
User asks real-world questions about timing, conflict, reactions, communication, family dynamics.

AI RESPONSE FORMAT (STRICT STRUCTURE):
1. Headline (One clear summary sentence)
2. Signal (Low Pressure, Medium Pressure, High Sensitivity)
3. Confidence (Overall, Data, Pattern)
4. What’s Happening (2–4 short bullet points)
5. Do This Now (One immediate action)
6. One Line To Say (One short sentence user can use)
7. Optional Expanders (More Context, Structural Details, Show Data, Visualize)

Safety Behavior:
* Tier 1 (Elevated): Suggest pause, lower tone.
* Tier 2 (Harm intent): De-escalate, encourage space, outside support.
* Tier 3 (Severe risk): Provide crisis resources.
Never validate harmful action. Never escalate.

## 9. DATA STORAGE & SECURITY

* Audit logging
* Token expiration
